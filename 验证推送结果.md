## ❌ 如果企业微信没收到消息

### 可能原因 1：应用可见范围未设置

**检查步骤**：
1. 访问企业微信管理后台：https://work.weixin.qq.com/
2. 应用管理 → 找到 "TrendRadar热点推送" 应用
3. 查看可见范围是否包含你的账号
4. 如果没有，添加你的账号

### 可能原因 2：用户 ID 不正确

**检查步骤**：
1. 打开企业微信 App
2. 我 → 编辑 → 查看账号
3. 确认账号是 `KuangXin`
4. 如果不是，更新 GitHub Secret `WEWORK_USER_ID`

### 可能原因 3：Secrets 配置错误

**检查步骤**：
1. 访问：https://github.com/wawojx-lab/TrendRadar/settings/secrets/actions
2. 确认以下 Secrets 配置正确：
   - WEWORK_CORP_ID
   - WEWORK_AGENT_ID
   - WEWORK_AGENT_SECRET
   - WEWORK_USER_ID

### 可能原因 4：时间不在推送时间段

**说明**：
- 工作流每小时运行一次（采集数据）
- 但只在早上 8:00-9:00 和晚上 8:00-9:00 推送消息
- 其他时间只采集数据，不推送

**解决方法**：
- 等待到推送时间段（早 8 点或晚 8 点）
- 或手动修改 `config/timeline.yaml` 调整推送时间
