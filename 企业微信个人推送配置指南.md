# 企业微信推送到个人微信 - 完整配置指南

## 🎯 目标
使用企业微信应用，将 TrendRadar 的热点新闻直接推送到你的个人微信。

---

## 📋 前置要求

### 必需条件
1. ✅ 企业微信管理员权限（或请管理员帮忙配置）
2. ✅ 企业微信账号
3. ✅ GitHub 账号

### 工作流程
```
GitHub Actions → 企业微信应用 → 你的个人微信
```

---

## 🔧 配置步骤

### 第一步：创建企业微信应用

#### 1.1 登录企业微信管理后台
- 访问：https://work.weixin.qq.com/
- 使用管理员账号登录

#### 1.2 创建自建应用
1. 点击 **"应用管理"**
2. 点击 **"自建"** 标签
3. 点击 **"创建应用"**
4. 填写应用信息：
   - **应用名称**：`TrendRadar热点推送`
   - **应用logo**：上传一个图标（可选）
   - **应用介绍**：`热点新闻自动推送`
5. 点击 **"创建应用"**

#### 1.3 获取应用信息
创建成功后，在应用详情页面获取：
- **AgentId**：类似 `1000002` 的数字
- **Secret**：点击"查看"获取，类似 `abc123xyz...`

#### 1.4 设置应用可见范围
1. 在应用详情页面
2. 找到 **"可见范围"**
3. 点击 **"添加"**
4. 选择 **"成员"**
5. 添加你自己（或所有需要接收推送的人）
6. 点击 **"确定"**

---

### 第二步：获取企业信息

#### 2.1 获取企业 CorpID
1. 点击左侧菜单 **"我的企业"**
2. 在企业信息页面找到 **"企业ID（CorpID）"**
3. 复制保存（类似 `ww1234567890abcdef`）

#### 2.2 获取通讯录管理 Secret（可选）
如果需要自动获取用户 ID，需要通讯录管理权限：
1. 点击 **"管理工具"** → **"通讯录同步"**
2. 点击 **"查看"** 获取 Secret
3. 如果没有权限，可以跳过这一步，手动获取用户 ID

---

### 第三步：获取你的用户 ID

#### 方法一：通过企业微信 App（推荐）
1. 打开企业微信 App
2. 点击右下角 **"我"**
3. 点击头像进入个人信息页面
4. 点击 **"编辑"**
5. 找到 **"账号"** 字段，这就是你的用户 ID
   - 格式类似：`zhangsan` 或 `ZhangSan` 或 `123456`

#### 方法二：通过管理后台
1. 管理后台 → **"通讯录"**
2. 找到你的名字
3. 点击进入详情
4. 查看 **"账号"** 字段

---

### 第四步：配置 Webhook URL

企业微信应用推送需要构建特殊的 Webhook URL：

#### 4.1 理解企业微信推送机制

企业微信应用推送有两种方式：

**方式 A：使用 Webhook（推荐，TrendRadar 支持）**
- 需要创建一个群机器人
- 但消息会推送到群，不是个人

**方式 B：使用应用消息 API（需要修改代码）**
- 直接推送到个人
- 需要调用企业微信 API
- TrendRadar 当前代码不支持

#### ⚠️ 重要发现

经过分析 TrendRadar 的代码，我发现：

**当前 TrendRadar 的企业微信实现只支持 Webhook 方式**，也就是群机器人模式。

虽然配置文件中有 `msg_type: "text"` 选项，但这只是改变消息格式（去除 markdown），并不能实现推送到个人微信。

## 📝 企业微信群机器人配置步骤（推荐方案）

### 1. 创建群聊
- 在企业微信中创建群聊
- 只邀请你自己（或添加几个需要接收推送的人）

### 2. 添加群机器人
- 群设置 → 群机器人 → 添加机器人
- 机器人名称：`TrendRadar热点推送`
- 复制 Webhook URL（格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxx`）

### 3. 配置 GitHub Secrets
```
WEWORK_WEBHOOK_URL = 你的 Webhook URL
```

### 4. 修改 config.yaml
```yaml
notification:
  channels:
    wework:
      webhook_url: ""  # 留空，使用 GitHub Secrets
      msg_type: "markdown"  # 群机器人使用 markdown
```

### 5. 推送到 GitHub
```bash
git add .
git commit -m "配置企业微信群机器人推送"
git push
```
