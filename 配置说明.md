## 🔴 需要你完成的配置

### 步骤 1：配置推送渠道（企业微信）

TrendRadar 的"个人微信推送"需要通过企业微信实现。

#### 方式一：企业微信群机器人（推荐）

1. **创建企业微信群**
   - 在企业微信中创建一个群聊（可以只有你自己）
   - 或者使用现有的群

2. **添加群机器人**
   - 群设置 → 群机器人 → 添加机器人
   - 自定义机器人名称（如"TrendRadar热点推送"）
   - 复制 Webhook 地址（格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxx`）

3. **配置到 GitHub Secrets**
   - 进入你的 GitHub 仓库
   - Settings → Secrets and variables → Actions
   - 点击 "New repository secret"
   - Name: `WEWORK_WEBHOOK_URL`
   - Value: 粘贴你的 Webhook 地址
   - 点击 "Add secret"

#### 方式二：企业微信应用（直接推送到个人）

需要创建企业微信应用，更复杂一些。建议先用群机器人。

### 步骤 2：配置 AI API Key

你需要提供 AI 服务的 API Key 才能使用 AI 分析功能。

#### 推荐使用 DeepSeek（性价比高）

1. **注册 DeepSeek 账号**
   - 访问：https://platform.deepseek.com/
   - 注册并登录

2. **获取 API Key**
   - 控制台 → API Keys → 创建新的 API Key
   - 复制 API Key（格式：`sk-xxxxx`）

3. **配置到 GitHub Secrets**
   - Name: `AI_API_KEY`
   - Value: 粘贴你的 API Key

#### 其他 AI 服务商（可选）

如果你有其他 AI 服务的 API Key，也可以使用：

- **OpenAI**：`openai/gpt-4o` 或 `openai/gpt-4o-mini`
- **Google Gemini**：`gemini/gemini-2.5-flash`
- **Anthropic Claude**：`anthropic/claude-3-5-sonnet`

需要同时配置：
- `AI_API_KEY`：你的 API Key
- `AI_MODEL`：模型名称（如 `openai/gpt-4o`）

1. 进入你的 GitHub 仓库
2. 点击 "Actions" 标签
3. 如果提示启用 Actions，点击 "I understand my workflows, go ahead and enable them"
4. 找到 "Get Hot News" 工作流
5. 点击 "Enable workflow"

## 🔧 常见问题

### Q1: 没有收到推送？

检查以下几点：
1. GitHub Secrets 是否正确配置
2. GitHub Actions 是否成功运行（查看 Actions 日志）
3. 企业微信机器人 Webhook 是否有效
4. 时间是否在配置的时间段内（8:00-9:00 或 20:00-9:00）

### Q2: AI 分析没有生效？

检查：
1. `AI_API_KEY` 是否正确配置
2. API Key 是否有效（余额是否充足）
3. 查看 Actions 日志中的错误信息

### Q3: 想修改推送时间？

编辑 `config/timeline.yaml` 文件中的 `custom.periods` 部分：
- `morning_push.start/end`：早间推送时间
- `evening_push.start/end`：晚间推送时间

### Q4: 想添加/删除关注领域？

编辑 `config/ai_interests.txt` 文件，修改兴趣描述。

### Q5: GitHub Actions 试用期到期？

GitHub Actions 试用版每 7 天需要签到一次：
1. 进入 Actions 页面
2. 找到 "Check In" 工作流
3. 点击 "Run workflow"
4. 试用期会重置为 7 天

**长期使用建议**：部署 Docker 版本（无时间限制）
