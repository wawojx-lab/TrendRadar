## 🔍 如果还是没收到消息

### 检查清单

```
□ 代码已推送成功
□ GitHub Actions 运行成功
□ 运行日志显示 "企业微信应用消息发送成功"
□ 企业微信应用可见范围包含你的账号
□ 企业微信收到消息
```

### 如果日志显示错误

**错误 1：获取 access_token 失败**

```
❌ 获取 access_token 失败：invalid corpid
```

**解决方法**：
- 检查 `WEWORK_CORP_ID` 是否正确
- 访问企业微信管理后台确认 CorpID

**错误 2：消息发送失败**

```
❌ 企业微信应用消息发送失败：no privilege
```

**解决方法**：
- 检查应用可见范围是否包含你的账号
- 检查 `WEWORK_USER_ID` 是否正确

**错误 3：用户不存在**

```
❌ 企业微信应用消息发送失败：user not found
```

**解决方法**：
- 确认 `WEWORK_USER_ID` 是你的企业微信账号
- 打开企业微信 App → 我 → 编辑 → 查看账号
