GitHub 需要几分钟来识别新推送的工作流文件。

**如果看到以下情况，请等待几分钟**：
- 页面显示 "Get started with GitHub Actions"
- 没有看到 "Get Hot News" 工作流
- 只看到 "Dependency Graph"

**刷新页面**，直到看到以下工作流：
- ✅ Get Hot News
- ✅ Clean old data
- ✅ Docker Build and Push
- ✅ Issue Guard

---

### 第二步：启用 GitHub Actions

如果看到提示：
```
Workflows are disabled by default for security reasons.
```

**操作**：
1. 点击 **"I understand my workflows, go ahead and enable them"**
2. 或者点击 **"Actions"** 标签右侧的 **"General"**
3. 在 "Actions permissions" 部分，选择 **"Allow all actions and reusable workflows"**
4. 点击 **"Save"**

---

### 第三步：手动触发测试

**方法一：通过网页界面（推荐）**

1. 访问：https://github.com/wawojx-lab/TrendRadar/actions
2. 点击左侧的 **"Get Hot News"** 工作流
3. 点击右侧的 **"Run workflow"** 下拉按钮
4. 确认分支是 `master`
5. 点击绿色的 **"Run workflow"** 按钮

**方法二：通过命令行**

```bash
gh workflow run "Get Hot News"
```

---

### 第四步：查看运行状态

**访问运行页面**：
```
https://github.com/wawojx-lab/TrendRadar/actions
```

你会看到：
- 🟡 黄色圆圈：正在运行
- ✅ 绿色对勾：运行成功
- ❌ 红色叉号：运行失败

**点击正在运行的工作流**，查看实时日志。

---

### 第五步：检查运行结果

#### 如果运行成功（绿色对勾）

1. **查看运行日志**
   - 点击工作流运行记录
   - 查看每个步骤的日志
   - 确认 "Run crawler" 步骤成功

2. **检查企业微信**
   - 打开企业微信 App
   - 查看是否收到来自 "TrendRadar热点推送" 应用的消息
   - 消息应该包含热点新闻和 AI 分析

3. **查看生成的报告**
   - 访问：https://github.com/wawojx-lab/TrendRadar/actions
   - 点击工作流运行记录
   - 在 "Artifacts" 部分下载报告（如果有）

#### 如果运行失败（红色叉号）

1. **查看错误日志**
   - 点击失败的工作流运行记录
   - 点击失败的步骤
   - 查看错误信息

2. **常见错误及解决方法**

   **错误 1：Secrets 未配置或配置错误**
   ```
   Error: WEWORK_CORP_ID is not set
   ```
   **解决方法**：检查 Secrets 配置是否完整

   **错误 2：企业微信认证失败**
   ```
   Error: 获取 access_token 失败
   ```
   **解决方法**：
   - 检查 `WEWORK_CORP_ID` 是否正确
   - 检查 `WEWORK_AGENT_SECRET` 是否正确
   - 确认应用可见范围包含你的账号

   **错误 3：AI API 调用失败**
   ```
   Error: AI API request failed
   ```
   **解决方法**：
   - 检查 `AI_API_KEY` 是否正确
   - 检查 `AI_API_BASE` 是否正确
   - 检查讯飞星火账户余额
